## text colors

- `text-gray-800` - Main headings and titles
- `text-gray-500` - Descriptions and helper text  
- `text-gray-600` - Labels and secondary text
- `text-gray-900` - Important values and amounts
- `text-white` - Text on colored backgrounds
- `text-blue-500` - Interactive elements and links
- `text-gray-700` - Body text and form labels
- `text-xs` - Small text size (12px)
- `text-sm` - Standard text size (14px)
- `text-red-600` - Expense amounts and errors
- `text-green-600` - Income amounts and success
